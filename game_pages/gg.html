Total Dev Time: Too long

 Believe it or not, this is the 2nd project I started working on, maybe a week after I finished Typey Jose... a lot has happened since then. Jams, schoolwork, and many many other things have delayed the development of this game (not to mention pure, utter laziness)

This is my most ambitious project to date. The premise is simple, you go to a grocery store, and you've gotta buy groceries. The problem: this grocery store is packed with characters that want this to not happen.

This game features a lot of animation, a lot of dialogue, and a lot of groceries. Coming soon to a new ground near you
